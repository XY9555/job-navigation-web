# AI密钥配置指南

## 智谱AI密钥获取

### 1. 注册智谱AI账号
- 访问：https://open.bigmodel.cn/
- 注册并完成实名认证

### 2. 获取API密钥
- 登录后进入控制台
- 点击"API密钥"
- 创建新的API密钥
- 复制密钥到 `backend/.env` 文件

### 3. 配置环境变量
```bash
# 在 backend/.env 文件中配置
ZHIPU_API_KEY=你的智谱AI密钥
AI_PROVIDER=zhipu
```

## 其他AI服务配置

### OpenAI

OPENAI_API_KEY=sk-你的OpenAI密钥

## 密钥过期解决方案

### 智谱AI密钥过期原因：
1. **免费额度用完** - 每月有调用次数限制
2. **密钥有效期** - 部分密钥有时间限制
3. **账户状态异常** - 需要重新验证

### 解决方法：
1. **重新生成密钥**
   - 登录智谱AI控制台
   - 删除旧密钥，创建新密钥
   - 更新 `.env` 文件

2. **充值续费**
   - 在控制台查看余额
   - 根据需要充值

3. **使用备选方案**
   - 系统已内置智能分析功能
   - 无需AI密钥也能提供基础评测

## 注意事项

1. **密钥安全**
   - 不要将密钥提交到代码仓库
   - 定期更换密钥

2. **成本控制**
   - 监控API调用次数
   - 设置合理的调用限制

3. **备选方案**
   - 系统支持多种AI服务
   - 可随时切换AI提供商
   - 内置智能分析确保功能可用
