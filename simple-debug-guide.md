# 📱 简化版APP调试指南

## 方法一：使用Chrome远程调试（推荐）

### 步骤：
1. **手机连接电脑**（USB线）
2. **手机开启USB调试**（按上面步骤操作）
3. **电脑打开Chrome浏览器**
4. **在地址栏输入**：`chrome://inspect`
5. **点击"Discover USB devices"**
6. **在手机上打开您的APP**
7. **在Chrome中会显示您的APP，点击"inspect"**
8. **在弹出的开发者工具中查看Console日志**

## 方法二：使用Android Studio（如果已安装）

### 步骤：
1. **打开Android Studio**
2. **底部点击"Logcat"标签**
3. **连接手机**
4. **选择您的设备**
5. **在过滤器中输入**：`chromium` 或 `WebView`
6. **打开APP查看日志**

## 方法三：手机端查看（最简单）

### 如果您的手机支持：
1. **打开APP**
2. **长按屏幕空白处**
3. **如果弹出菜单，选择"检查"或"调试"**
4. **查看控制台输出**

## 关键日志信息

### 需要关注的日志：
- `🌐 API请求详情`
- `📡 API响应状态`
- `❌ API请求错误详情`
- `🔍 测试网络连接`
- `⚠️ 移动端网络连接异常`

### 常见错误类型：
1. **网络连接失败**：`Failed to fetch`
2. **CORS错误**：`CORS policy`
3. **超时错误**：`timeout`
4. **DNS解析失败**：`DNS resolution failed`
5. **SSL证书错误**：`SSL certificate`

## 快速测试方法

### 在APP中测试：
1. **打开APP**
2. **尝试注册新用户**
3. **观察是否有网络错误提示**
4. **如果有错误，记录具体的错误信息**

### 告诉我的信息：
- 错误发生在哪个功能（注册/登录/其他）
- 具体的错误提示文字
- 是否能看到任何网络请求
- 手机的网络环境（WiFi/4G/5G）
