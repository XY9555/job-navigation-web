## 🔧 快速启动指南

### 1. 启动开发环境
```bash
# 运行启动脚本
start-development.bat

# 或手动启动
cd backend && npm run dev  # 后端
npm run dev                # 前端
```

### 2. 构建Android应用
```bash
# 运行构建脚本
build-android-app.bat

# 或手动构建
npm run build
npx cap sync android
npx cap open android
```

### 3. 安装到手机
1. 构建APK: `android/gradlew.bat assembleDebug`
2. 找到APK: `android/app/build/outputs/apk/debug/app-debug.apk`
3. 传输到手机并安装
