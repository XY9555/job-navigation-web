# 部署到云端服务器指南

## 🚀 推荐方案：使用免费云服务

### 1. 后端部署选择（免费）
- **Render.com** (推荐) - 免费500小时/月
- **Railway.app** - 免费$5额度/月  
- **Vercel** - 适合API部署
- **Heroku** - 有免费层

### 2. 数据库选择（免费）
- **PlanetScale** - 免费MySQL数据库
- **Supabase** - 免费PostgreSQL + 实时功能
- **MongoDB Atlas** - 免费512MB
- **Render PostgreSQL** - 免费90天

### 3. AI服务
- 继续使用智谱AI（已配置）
- 或切换到其他AI服务

## 📋 部署步骤

### 第一步：准备云端数据库
1. 注册Supabase账号（推荐）
2. 创建新项目
3. 获取数据库连接信息
4. 修改后端数据库配置

### 第二步：部署后端到Render
1. 将代码推送到GitHub
2. 连接Render到GitHub仓库
3. 配置环境变量
4. 自动部署

### 第三步：修改APP配置
1. 更新API地址为云端地址
2. 重新构建APP
3. 测试所有功能

## 💰 成本估算
- 开发阶段：完全免费
- 生产环境：$0-20/月（根据用户量）
