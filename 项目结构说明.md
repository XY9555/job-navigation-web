## 🎯 核心功能模块

### 1. 用户认证系统
- **注册/登录**: JWT token认证
- **用户管理**: 个人信息、头像、设置
- **安全**: 密码加密、token验证

### 2. 简历管理系统
- **CRUD操作**: 创建、读取、更新、删除简历
- **数据结构**: 个人信息、工作经历、教育背景、技能、项目
- **状态管理**: 草稿、发布、归档

### 3. AI智能分析
- **简历评测**: 智谱AI分析简历质量
- **职位匹配**: AI分析简历与职位匹配度
- **面试问题**: 生成个性化面试问题
- **优化建议**: 提供简历改进建议

### 4. 移动应用
- **跨平台**: Capacitor支持Android/iOS
- **响应式**: 适配各种屏幕尺寸
- **原生功能**: 状态栏、键盘、触觉反馈

## 🔧 技术栈

### 前端技术
- **Vue.js 3**: 渐进式JavaScript框架
- **Vite**: 快速构建工具
- **Vue Router**: 单页应用路由
- **Axios**: HTTP客户端
- **Vant**: 移动端UI组件库

### 后端技术
- **Node.js**: JavaScript运行环境
- **Express.js**: Web应用框架
- **SQLite**: 轻量级数据库
- **Sequelize**: ORM数据库操作
- **JWT**: 用户认证

### 移动端技术
- **Capacitor**: 跨平台移动应用框架
- **Android Studio**: Android开发环境
- **Gradle**: Android构建工具

### AI服务
- **智谱AI**: GLM-4大语言模型
- **API集成**: RESTful API调用
- **数据处理**: JSON格式数据交换

## 🚀 快速开始

### 1. 环境准备
```bash
# 确保已安装 Node.js 16+
node --version

# 确保已安装 Android Studio (可选)
```

### 2. 配置数据库
```bash
# 自动配置SQLite数据库
setup-database.bat
```

### 3. 启动开发环境
```bash
# 启动前后端服务
start-development.bat
```

### 4. 构建移动应用
```bash
# 构建Android APK
build-android-app.bat
```

### 5. 测试功能
```bash
# 运行完整功能测试
node test-complete.js
```
